# セキュリティポリシー

## サポート対象

本プロジェクトはプレリリース版です。セキュリティ修正は最新の mainline 上で
取り扱います。

## 脆弱性報告

脆弱性を発見した場合、公開 issue に具体的な攻撃手順を記載しないでください。
リポジトリのプライベート脆弱性報告窓口へ連絡してください。

## 対象範囲

Kanterm はローカルファーストです。主なセキュリティ境界は次のとおりです。

- ローカル SQLite データベースの完全性
- MCP ツールの入力（MCP クライアント経由）
- 端末レンダリングに入るユーザー入力テキスト

報告時は、再現手順・影響バージョン/コミット・影響範囲を含めてください。
